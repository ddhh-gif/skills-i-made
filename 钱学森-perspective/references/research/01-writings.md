## 方法论 A：系统工程 / 总体设计部（皇冠明珠，最可执行）

**定义（钱学森原话，p.6）**：「系统工程就是从系统的认识出发，设计和实施一个整体，以求达到我们所希望得到的效果。」称「工程」是要强调——**要达到效果、要具体、要有可行措施、要实干、要改造客观世界**。不加「学」字，就是强调它是实践、要落地。

**触发条件**：一个目标复杂到「一个总工程师 + 几个副总，已经抓不住总体设计」。美国造原子弹1.5万人、登月42万人——靠一个人不可能（p.15-16）。

**核心操作：设总体设计部**（p.14-16）
- 不是几个人、十几个人，常常几百人甚至近千人的组织，做领导的「技术参谋部门」。
- 职责：把复杂体系各组成部分协调好，使最终整体达到要求的性能；指挥各部分「怎么设计，使联系起来的整体更好地工作」。
- 类比链（他本人用的）：单个泥瓦匠盖房=设计师+工人合一 → 工场协作需「监工」 → 大工业需「总工程师」 → 现代大系统需「总体设计部」。乐队类比：一个提琴手自己指挥自己，一个乐队需要指挥。

**配套操作1：六要素分析法**（p.17-18）——拆解任何「系统/分系统」
任何系统都由六要素组成：**人**（第一要素）+ **物**（物资/设备/财）+ **事**（任务指标/信息）。
- 分析一个组织 = 追踪这六要素的「流通」。各分系统的相互作用，通过六要素流通体现。
- 信息是关键要素：系统越大越复杂，信息相对增长越大；「信息反馈失灵就会导致管理混乱」。
- 可迁移：分析任何工厂/学校/部门，先列这六要素、画出物流与信息流。

**配套操作2：约束下求总体最优**（p.19）
1. 认识制约：分两类——经济规律的制约 + 技术条件的制约。
2. 据制约画网络图（PERT/计划协调技术，1958美国北极星导弹首用，1962中国国防试用，p.10/17）。
3. 求「主要矛盾线」（关键路径）。
4. 在制约下求总体最优——「在制约下求得总体最优是企业经营管理的一个重要概念」。

**配套操作3：定量化铁律**（p.23-27，反复强调，真信念）
- 凡规划必须定量，不能「走着瞧、走一步看一步」，要「看几步」。
- 把数量关系归纳成数学方程组（数学模型）→ 在约束条件下求解 = 「系统的数学模拟」（p.9-10）。
- 例（炼钢，p.26）：炼一吨钢要多少铁矿/电力/焦煤/石灰/人力？把关系全部定量表达 → 上计算机 → 算多个方案（十个、二十个）→ 比较哪个好。
- **可验算**：怕算法靠不住，就「往回算」——从今年往回算去年，去年情况已知可比较。
- 社会科学做不了实物实验，但「可以在电子计算机上试验」。

**普适性（关键——证明这是通用方法而非航天专用）**：
- 小：设计一个不大的设备也要考虑部件协调；治病要「人、病、症三结合，以人为主统筹」，把人体+环境当复杂体系（p.7, p.20）。
- 中：工厂经营管理、一所大学（万把人、十几个系，复杂度不亚于工厂）= 教育系统工程（p.21）。
- 大：整个国家建设规划 = 社会工程（社会系统工程），用运筹学+控制论+计算机预见社会发展（p.23-27）。
- 每个领域是一个「专业」（工程/军事/经济/社会/教育/环境/法治…），有特有学科基础；但**共同基础 = 运筹学 + 控制论 + 信息论 + 计算技术**（p.9-10）。

## 方法论 B：基础理论当筛子（判断真伪/选路，p.242-243）

研究 = 在没走过的路上摸索，但「摸索不能盲目」。用自然界一般规律（物理化学）当指南针：
- 在许多看似可走的路中，分辨**哪条肯定对 / 哪条可能对 / 哪条肯定错**。
- 不在「肯定错」的路上花功夫，集中精力在「肯定是或可能是对」的路上。
- 实例（直接拿来用的证伪规则）：永动机不可能 → 一切含永动机构的机器不必想；量子力学共生量不可同时精确测定 → 要求同时知道位置和动量的微观理论是错的；化学键能量已知 → 「发明了比汽油能量大一倍的碳氧燃料」必是错的，不必信。
- 要点：不是「知道」规律，而是「充分掌握、里里外外摸透」，才能在复杂事物中**分析出核心问题**、不被假象蒙蔽。

## 方法论 C：科学工作的「手艺」/基本功（p.245-249）

「一个科学技术工作者也要像工人一样讲究手艺」。科学是严肃、严格、严密、不允许马虎的。
- **有条有理**：书写清楚；符号系统不能乱换；理论推算和实验有条理写下；记录要保存、要有档案（「日后可查」）。
- **理论操作**：推理锐利、不拖泥带水；分清「什么可能、什么一时还不清楚」；运算敏捷准确——记熟常用数学关系（三角/微积分公式）+ 用得熟。「这虽都是死功夫，但非常重要」。靠多做习题练。
- **实验操作**：①做实验「不在做得多，而在做得少」——少而精，事先研究测量方法/仪器/误差分析，否则大部分数据没价值；②眼明手快、严守操作规范、不随便「别出心裁」。
- 单位错误警告（p.249）：「公里」误写「米」，答案差一千倍。

## 方法论 D：从薄到厚，再从厚到薄（学习/吃透一门学问，华罗庚说法，钱赞同，p.250）

- **从薄到厚**：初读觉得简单，深入后发现问题复杂、需学的越来越多——表示书里的问题在脑子里「展开」了，便于逐个解决。
- **从厚到薄**：分清「什么是最基本理论 / 什么是派生 / 什么是第三次推出」；分清基本概念、概念间关系、谁主谁次、每个概念的来源、与事实的关系、**局限性**（什么时候代表事实、什么时候不能）。搞清这些 → 不必记大量内容，只「提纲挈领」装脑子。
- **吃透的标志（可检验）**：拿一篇没读过的论文，看第一页（什么问题/作者看法/什么方法）→ 因彻底掌握学科，立刻能估计「这方法对不对头、该导致什么结果」→ 看结论对比，相差很远则论文很可能有问题。

## 方法论 E：塔尖原理 / 严格自我批判（p.277）

- 论文发表的只是「搞对」的一小部分——常常是全部工作量的 1/10 甚至 1/100，错的 9/10 锁在抽屉。
- 「正确的结果，是从大量错误中得出来的；没有大量错误做台阶，也就登不上最后正确结果的高座。」
- 刚开始时错的可能性总大于对的；要勇于认错，毫不可惜地扔掉「自己辛辛苦苦花心血得来的」错东西。缺乏认错精神会吃大亏。
- 自证：薄壳论文发表只几十页，反复演算报废 700 多页。

## 方法论 F：现代科学技术体系 = 问题定位坐标系（p.204-210）

九大部门：自然科学/社会科学/数学科学/系统科学/思维科学/人体科学/文艺理论/军事科学/行为科学。最高概括是马克思主义哲学（辩证唯物主义），每部门通过一架「桥梁」连到哲学（自然科学↔自然辩证法，系统科学↔系统论，思维科学↔认识论…）。每部门（除文艺理论）分三层次：**基础科学 → 技术科学（应用科学）→ 工程技术**。

可执行用法：
1. **同对象多侧面法**（p.204）：每个部门都研究「整个客观世界」，只是侧面不同。同一个工厂，自然科学家看机器/能量，社会科学家看财政/管理。→ 拆任何问题时，强制从多个侧面（物质运动面/社会运动面/系统结构功能面/人的认识面…）各看一遍。
2. **三层次定位**：任何知识/技术先定位在基础科学/技术科学/工程技术哪一层。技术科学是「桥」——理论联系实际的关键层（钱本人就是技术科学家自居）。
3. **唯象理论台阶**（p.209）：从经验/现象上升到科学的中间层。先把现象「理出条理」成概括性经验学问（知其然，如 PV=CT 气体定律），再追求底层解释（知其所以然，如统计物理）。承认唯象理论是必要的中间层，别急于跳过。
4. **前科学→科学**（p.208-209）：经验（只知其然，如中医）= 前科学，不等于不重要；但要纳入体系、能说清「为什么」、与其他部门融合一致，才算科学。体系是**开放**的——对古今中外先甄别、好的吸收、坏的抛弃。

## 方法论 G：创造 = 逻辑+形象+灵感三种思维交错（p.143-148, 谈思维科学）

- 人能控制的思维分三种：**抽象（逻辑）思维、形象（直感）思维、灵感（顿悟）思维**。划分是为研究方便，实际每个思维过程都是两三种交错。
- 「人的创造思维过程就绝不是单纯的抽象（逻辑）思维，总要有点形象（直感）思维，甚至要有灵感（顿悟）思维。」→ 纯逻辑推不出创造。
- **可执行推论**：要提升创造力，必须养形象思维——而形象思维靠艺术/美学训练（p.69-70：美感是思维过程的结果；科学家需从文艺吸取营养）。这是钱「科学家要懂艺术」主张的方法论根，不是鸡汤。
- **集体思维/社会思维学**（p.147-148）：人的思维是集体的，靠社会实践+前人知识。学术讨论的价值——「讲错话、提错误意见的人，对最后得出的正确结论也是有贡献的」。鹅湖之会（1175）规定「各种意见都可以讲，不同意老师也可以讲，老师不能骂学生」。→ 这是他晚年「综合集成研讨厅」（把专家经验+数据+模型经集体研讨从定性汇成定量）的种子。

## 反复出现≥3次的真信念（贯穿全书）
1. **定量、上计算机算、多方案比较**——出现在系统工程、社会工程、研究方法各处。
2. **理论必须联系实践 / 要改造客观世界 / 要「实干」**——「工程」二字的执念。
3. **整体大于部分之和，先看总体再看局部**——系统思想内核。
4. **去神秘化**：「系统工程的概念并不神秘」「未来学并不神秘」「计算与分析不是什么神秘的东西」——反复用「并不神秘」拆解高大上概念。
5. **马克思主义哲学（辩证唯物主义）为指导**——每个方法论都挂回哲学；这是时代烙印，也是他真实的思维框架（见诚实边界）。
